In other words, find the number of 4-element subsets of these points that can be written as {A, B, C, D} such that AB || CD, and BC || AD. No four points are in a straight line.Input

Input starts with an integer T (**≤ 15)**, denoting the number of test cases.

The first line of each test case contains an integer n (1 ≤ n ≤ 1000). Each of the next n lines, contains 2 space-separated integers x and y (the coordinates of a point) with magnitude (absolute value) of no more than 1000000000.Output

For each case, print the case number and the number of parallelograms that can be formed.Sample Input

分析：

平行四边形存在定理。

两根线段的中点在同一个位置，那么这两个线段就能组成一个平行四边形的一对对角线。

using namespace std;

const double PI = acos(-1.0);

const int MAXN = 1e3 + 10;
struct Poi
{
    int x, y;
    friend bool operator < (Poi A, Poi B)
    {
        if(A.x == B.x)  return A.y < B.y;
        else            return A.x < B.x;
    }
};

{
    int t;
    int n;
    cin >> t;
    for(int kase = 1; kase <= t; kase ++)
    {
        scanf("%d", &n);
        for(int i = 1; i <= n; i++)
            scanf("%d%d", &poi[i].x, &poi[i].y);
        int ind = 0;
        for(int i = 1; i < n; i++)
            for(int j = i + 1; j <= n; j++)
            {
                ind++;
                mid[ind].x = (poi[i].x + poi[j].x) ;        //有点东西哦
                mid[ind].y = (poi[i].y + poi[j].y) ;
            }
        sort(mid + 1, mid + 1 + ind);
        int cnt = 1;
        ll ans = 0;
        for(int i = 1; i < ind; i++)
        {
            if(mid[i].x == mid[i + 1].x && mid[i].y == mid[i + 1].y)
            {
                cnt ++;                                             //记录的是同一中点的数目
            }
            else
            {
                ans += cnt*(cnt - 1) / 2;
                cnt  = 1;
            }
        }
        printf("Case %d: %lld\n", kase, ans);
    }
    return 0;
}

LightOJ - 1095 Arrange the Numbers

Posted on 2019-10-03 | In 数论

Consider this sequence {1, 2, 3 … N}, as an initial sequence of first N natural numbers. You can rearrange this sequence in many ways. There will be a total of N! arrangements. You have to calculate the number of arrangement of first N natural numbers, where in first M positions; exactly K numbers are in their initial position.

For Example, N = 5, M = 3, K = 2

You should count this arrangement {1, 4, 3, 2, 5}, here in first 3 positions 1 is in 1st position and 3 in 3rd position. So exactly 2 of its first 3 are in there initial position.

But you should not count {1, 2, 3, 4, 5}.Input

Input starts with an integer T (**≤ 1000)**, denoting the number of test cases.

Each case contains three integers N (1 ≤ N ≤ 1000), M (M ≤ N), K (0 < K ≤ M).Output

For each case, print the case number and the total number of possible arrangements modulo 1000000007.Sample Input

2

5 3 2

10 6 3Sample Output

Case 2: 64320

分析：

黑板上排列组合，你舍得解开吗

两种思路：1、错排公式 2、容斥定理。

利用了组合数取模。利用的是扩展欧几里得求逆元。

1、错排公式

dp[n] 表示的是将n个元素错排的方法数

可知 dp[n] = (n-1)dp[n-2] + (n-1)dp[n-1];

dp[0] = 1; dp[1] = 0; dp[2] = 1; dp[3] = 2;

2、容斥定理（我自己推的，太难写了，略）

    4. 模块化答题

    可以用错排公式求解，也可以用容斥定理推（自己推出来啦）
    用递推公式求错排数组
    dp[n] 表示的是将n个元素错排的方法数
    可知
    dp[n] = (n-1)dp[n-2] + (n-1)dp[n-1];
    //见收藏的博客
    显然dp[2] = 1;
        dp[3] = 2;
    所以我们枚举后面 n - m个数中需要错排的数目i [0, n - m]
    与前面必须要错排的 m - k个数凑在一起，一共要有 m - k + i 个元素进行错排
    将这些情况累加。最后与C(m,k)相乘即可


    C(N, M) = N!/M!/(N-M)!
*/
#include<functional>
#include<cstdio>
#include<cstring>

#define md(x, y)  (x + y) / 2
#define ls(x)      x << 1
#define rs(x)      x << 1 | 1
using namespace std;
typedef long long ll;
const double PI = acos(-1.0);
const double eps = 1e-6;
const int INF = 0x3f3f3f3f;
const int MAXN = 1e3 + 10;
const int MOD  = 1000000007;
ll dp[MAXN], ni[MAXN], fac[MAXN];
ll exgcd(ll a, ll b, ll &x, ll &y)
{
    if(b == 0)
    {
        x = 1;
        y = 0;
        return a;
    }
    else
    {
        ll temp = exgcd(b, a%b, y, x);
        y -= x * (a / b);
        return temp;
    }
}
void init()//考虑取模
{
    dp[0] = 1;
    dp[1] = 0;              //WA的原因
    dp[2] = 1;
    dp[3] = 2;
    for(int i = 4; i < MAXN; i++)
        dp[i] = ( (i - 1) * ((dp[i - 1] + dp[i - 2]) % MOD)) % MOD;

    //求出fac表和逆表
    fac[0] = 1;
    ni[0] = 1;
    ll x, y;
    for(int i = 1; i < MAXN; i++)
    {
        fac[i] = (fac[i - 1] * i) % MOD;
        exgcd(fac[i], MOD, x, y);
        ni[i] = (x % MOD + MOD) % MOD;
    }
}
ll C(int N, int M)
{
    return ((fac[N] * ni[M]) % MOD * ni[N - M]) % MOD;
}
int main()
{
    int t;
   // freopen("in.txt", "r", stdin);
    //freopen("out.txt", "w", stdout);

    int n, m, k;

    {
        scanf("%d%d%d", &n, &m, &k);
        ll ans = 0;
        for(int i = 0; i <= n - m; i++)
            ans = (ans + (C(n - m, i) * dp[m - k + i]) % MOD) % MOD;

        ans = (ans * C(m, k)) % MOD ;
        printf("Case %d: %lld\n", kase, ans);
    }

分析：

根据隔板法，我们知道答案就是C(n+k-1, k-1)

隔板法就是在n个元素间的（n-1）个空中插入k个板，可以把n个元素分成k+1组的方法。　　

应用隔板法必须满足3个条件： 　　
（1） 这n个元素必须互不相异；
（2） 所分成的每一组至少分得1个元素；
（3） 分成的组别彼此相异。

公式
把10个相同的小球放入3个不同的箱子，每个箱子至少一个，问有几种情况？

C（n-1，m-1）=C(9.2)

例2. 求方程 x+y+z=10的非负整数解的个数。

分析：注意到x、y、z可以为零，故例1解法中的限定“每空至多插一块隔板”就不成立了，怎么办呢？只要添加三个球，给x、y、z各添加一个球，这样原问题就转化为求 x+y+z=13的正整数解的个数了，则问题就等价于把13个相同小球放入3个不同箱子，每个箱子至少一个，有几种情况？易得解的个数为C（n+m-1,m-1）=C（12，2）=66（个）。

隔板法博客https://blog.csdn.net/sdz20172133/article/details/81431066

那么打逆元表的方法又有两种，一种是利用扩展欧几里得算法求逆元，一种是用费马小定理+快速幂求逆元。

唯一分解定理。、

我们将n进行分解

再得到 σ (n)的表达式。我们要想使得 σ （n）为偶数的情况显然很多，所以我们可以考虑使得其为奇数的情况，然后再用总数减去奇数的数目就是偶数的数目了。

我们知道 σ （n）是一堆数的乘积。我们知道，要想使得一堆数的乘积为奇数。那么这一堆数中的每一个数都必须是素数。

即(1 + pi + pi^2 + … + pi^ai) 这个数是奇数。

对于加减法，我们知道奇数加奇数是奇数，奇数加偶数是奇数。所以要想使得以上的式子为奇数

我们也可以知道，素数除了2之外都是奇数，所以我们这里先考虑p为奇数的情况

显然pi, pi ^ 2, pi ^ 3 ….. pi ^ ai 这ai个数都是奇数。我们知道要想使得奇数之和为偶数，就必须有偶数个奇数。 也就是ai是个偶数。 也就是可以提出一个2来，那么可以将他视为一个平方数。

所以[1, n]中的平方数，和2倍平方数的 σ 都是奇数。所以n减去这些数即可。
