HDU 1394 Minimum Inversion Number

Posted on 2019-09-13 | In 数据结构

The inversion number of a given number sequence a1, a2, …, an is the number of pairs (ai, aj) that satisfy i < j and ai > aj.

For a given sequence of numbers a1, a2, …, an, if we move the first m >= 0 numbers to the end of the seqence, we will obtain another sequence. There are totally n such sequences as the following:

a1, a2, …, an-1, an (where m = 0 - the initial seqence)
a2, a3, …, an, a1 (where m = 1)
a3, a4, …, an, a1, a2 (where m = 2)
…
an, a1, a2, …, an-1 (where m = n-1)

You are asked to write a program to find the minimum inversion number out of the above sequences.
InputThe input consists of a number of test cases. Each case consists of two lines: the first line contains a positive integer n (n <= 5000); the next line contains a permutation of the n integers from 0 to n-1.
OutputFor each case, output the minimum inversion number on a single line.
Sample Input

10
1 3 6 9 0 8 5 7 4 2

Sample Output

16

分析：

要清楚整个数列的逆序数代表的含义，每个数的逆序数的含义。

复习一下用树状数组解普通逆序数问题

普通求逆序数的思路，将数列排序离散化，然后按原数列顺序从后往前依次将数放入（以离散化后的下标），然后用线段树维护离散化后的数， 可用树状数组解。

然后对逆序数这个这个数的性质进行思考
第一个数的逆序数，就是在他之后的，比他小的数
把第一个数移到最后去，则他的逆序数变成了0.
那些比他大的数，他们的逆序数都加1

那些比他小的数，逆序数没有发什么变化。

而第一个数的逆序数，就是比他小的数的数目
也就是统计一下，比第一个数大的数有哪些
那么这一次移动，总的逆序数的变化
sum = sum - (第一个数的逆序数) + 数列中比第一个数大的数的数目

= sum - (数列中比第一个数小的数的数目) + (数列中比第一个数大的数的数目)
用树状数组解出总的逆序数，和每个点的逆序数。

代码：

#include<functional>
#include<cstdio>
#include<cstring>
#include<algorithm>
#include<iostream>
#include<string>
#include<vector>
#include<stack>
#include<bitset>
#include<cstdlib>
#include<cmath>
#include<set>
#include<list>
#include<deque>
#include<map>
#include<queue>
#define md(x, y) (x + y) / 2
using namespace std;
typedef long long ll;
const double PI = acos(-1.0);
const double eps = 1e-6;
const int INF = 0x3f3f3f3f;
const int MAXN = 5e3 + 10;
int inverNum[MAXN];
int q;
int c[MAXN], a[MAXN], lisan[MAXN];
int lowbit(int x)
{
    return x&(-x);
}
void update(int x, int v)
{
    for(int i = x; i <= q; i += lowbit(i))
        c[i] += v;
}
int sum(int x)
{
    int ans = 0;
    for(int i = x; i > 0; i -= lowbit(i))
        ans += c[i];
    return ans;
}
int binSearch(int x, int n)
{
    int l = 1;
    int r = n;
    while(l <= r)
    {
        int mid = md(l, r);
        if(lisan[mid] == x)
            return mid;
        else if(lisan[mid] > x)
            r = mid - 1;
        else
            l = mid + 1;
    }
}
int main()
{
    int n;
    while(scanf("%d", &n) != EOF)
    {
        for(int i = 1; i <= n; i++)
        {
            scanf("%d", &a[i]);
            lisan[i] = a[i];
        }
        sort(lisan + 1, lisan + 1 + n);
        q = unique(lisan + 1, lisan + 1 + n) - (lisan + 1);
        memset(c, 0, sizeof c);
        int ans = 0;
        for(int i = n; i > 0; i--)
        {
            int pos = binSearch(a[i], q);
            update(pos, 1);
            inverNum[i] = sum(pos - 1);
            ans += inverNum[i];
        }
        //第一次移第一个数，第二个以第二个数
        for(int i = 1; i < n; i++)
        {
            inverNum[i] = 0;
            int temp = 0;
            for(int j = 1; j <= n; j++)
                if(i != j && a[j] > a[i])
                    inverNum[j]++;
            for(int j = 1; j <= n; j++)
                temp += inverNum[j];

            ans = min(ans, temp);
        }
        printf("%d\n", ans);

    }
    return 0;
}
